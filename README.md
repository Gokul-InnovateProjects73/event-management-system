# EventHub - Event Management System

## Project Overview
The Event Management System (EventHub) is a web-based application designed to organize and manage college or organizational events efficiently. The system allows administrators to create, update, and manage various events such as workshops, seminars, technical fests, and cultural programs.

## Key Features
- **Event Management**: Create, update, and manage workshops, seminars, technical fests, and cultural programs.
- **Browse & Register**: Users can browse available events, view detailed information, and register their participation.
- **Centralized Database**: All event details (date, location, description, seat availability) are stored and managed centrally.
- **RSVP Options**: Participants can confirm attendance with "Attending", "Maybe", or "Not Attending".
- **Real-time Updates**: Live event schedules and countdown timers for upcoming events.
- **Admin Dashboard**: Monitor registrations and manage event capacity effectively.

## UI/UX Redesign
The user interface has been completely redesigned with a modern and professional layout. The updated design includes improved color themes, better navigation, responsive components, and a clean user-friendly interface. These enhancements provide a more attractive visual appearance and make it easier for students to view upcoming events and register their participation through the system.

Key improvements include:
- **Enhanced Usability**: Streamlined navigation and intuitive layouts.
- **Modern Aesthetics**: A clean, contemporary design focusing on visual clarity.
- **Responsive Design**: Optimized for a seamless experience across all devices.
- **Clarity & Focus**: Minimalist approach to help users find and register for events quickly.
- **Reliable Rendering**: Optimized component loading to prevent rendering errors and ensure a smooth initial load.

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js & Express.js
- **Database**: MongoDB
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Getting Started
### Prerequisites
- Node.js
- MongoDB

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   # In /server
   npm install
   # In /client
   npm install
   ```
3. Create a `.env` file in the `server` directory with:
   ```
   PORT=5000
   MONGO_URI=your_mongodb_uri
   JWT_SECRET=your_secret
   ```
4. Start the development servers:
   ```bash
   # In both /server and /client
   npm run dev
   ```

## Objective
The main objective of this project is to provide a user-friendly platform similar to KnowFest for managing and participating in events online, reducing manual processes and improving efficiency.
